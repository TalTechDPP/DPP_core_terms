# ESPR DPP Core terms
## The 'long list' of terms for ESPR-compliant DPP (eco)systems.

This repository provides the 'long list' of terms for ESPR-compliant DPP (eco)systems.
The 'long list' of terms for ESPR-compliant DPP (eco)systems is composed as a result of careful analysis of collaborative term classification and harmonisation with bottom-up conceptualization process. 

The third consolidated revision of this vocabulary provides a comprehensive vocabulary of key terms for EU DPP domain from the regulations annotated with definitions and references to regulatory acts in Web Ontology Language (OWL).
In this revision, if multiple definitions from regulatory acts were detected for a term, they were merged under one class. The alternative definitions are provided, whenever applicable, as dcterms:description annotations for each term.

The vocabulary is provided as is.
